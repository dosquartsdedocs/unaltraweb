This website is a reusable template for project-driven sites built with Jekyll and al-folio.
It keeps the technical pieces in place so each new project only needs to replace the content.

Use this template to:

- present project goals, methods, timeline, and partners,
- publish updates through the news and blog sections,
- list publications, repositories, and reusable outputs,
- maintain a consistent structure across all project websites.

The multilingual setup uses `lang` and `ref` page metadata, language folders under `_pages/`, and a browser-aware redirect on the root path.
